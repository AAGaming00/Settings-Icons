## Settings List Icons
Add neat looking icons to the powercord settings menu (works well with [this](https://canary.discordapp.com/channels/538759280057122817/755005803303403570/762067626942791680))

## Preview
![Preview](https://cdn.discordapp.com/attachments/756986970231668847/761772295726694420/unknown.png)

## Installation
If you'd like to import this snippet, go to your **Themes tab, open Quick CSS** and paste the following:
```
@import url("https://raw.githack.com/Snapperito/CSS-Snippets/main/Settings%20Icons/settingsicons.css");
```
